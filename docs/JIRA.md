# Jira Documentation

## Project Management

The Substracker project was managed using Jira to organize development activities and monitor project progress.

---

## Workflow

The project followed the workflow below:

1. Backlog
2. To Do
3. In Progress
4. Review
5. Done

---

## Task Categories

The following task categories were managed:

- UI Design
- Authentication Module
- Dashboard Module
- Subscription Management
- SQLite Database
- Documentation
- Testing

---

## Sprint Activities

The development process included:

- Requirement Analysis
- Interface Design
- Application Development
- Testing
- Final Review

---

## Benefits of Jira

Using Jira helped the team to:

- Track development progress
- Organize project tasks
- Improve collaboration
- Monitor task completion efficiently
