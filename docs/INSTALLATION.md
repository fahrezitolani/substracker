# Installation Guide

## Requirements

Before running the application, ensure the following software is installed:

- Flutter SDK
- Dart SDK
- Android Studio
- Visual Studio Code
- Git

---

## Clone Repository

```bash
git clone https://github.com/fahrezitolani/substracker.git
```

---

## Open Project

```bash
cd substracker
```

---

## Install Dependencies

```bash
flutter pub get
```

---

## Run Application

```bash
flutter run
```

---

## Build APK

```bash
flutter build apk
```

---

## Notes

Ensure Android SDK and Flutter SDK are properly configured before running the application.
